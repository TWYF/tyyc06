　1A指定花式為了培養參賽者的穩定度，以及提升選手們的層次，因此本屆選定下列指定動作
　讓參賽選們完成。
　※可使用Bind Returns回收
 
　1. Black Hop
　　重點提示： 跳躍放線及上線，不得使用拇指輔扣住線。
　2. Eli Hops X 5
　　重點提示： 往上彈跳雙手需碰到。
　3. Iron Whip X 3
　　重點提示： 繩圈套上後線不得交錯。
　4. Kwijibo X 1
　　重點提示： 跳躍時Yo-Yo高度必需過頭。
　5. Shock Wave X 3
　　重點提示： 左右一次為一個循環，如影片。
　6. Slack Trapeze
　　重點提示： 以影片為標準，繞繩時左手拇指不得扣住線，左撇子為右手拇指。
　7. Spirit Bomb
　　重點提示： 第二次跳躍時進入右手弧口與線的交錯洞內。左撇子為左手弧口。
　8. Suicide X 3
　　重點提示： 限定用食指接住。
　9. Three Seven
　　重點提示： 每一次各作七下。

　10. White Buddha　( 待補上完整版 )
　　 重點提示： 最後動作必需卡住不得掉下。
